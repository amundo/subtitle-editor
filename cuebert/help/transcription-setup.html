Transcription Setup Guide

Use the prepared Cuebert transcription kit to make a first-draft transcript, then open the JSON file in Cuebert for correction.

What You Need

  1. The Cuebert transcription kit folder.
  2. The audio or video file you want to transcribe.
  3. Cuebert for reviewing and correcting the transcript.
No Homebrew, Deno, Git, Xcode, or whisper.cpp setup is needed if the kit has already been prepared.

Check the Kit

Open Terminal and run: 

cuebert-transcribe --setup-check

If the kit is ready, it will show the whisper.cpp program and model it found.

Transcribe One File

Put the media file somewhere easy to find, such as the Desktop. Then run: 

cuebert-transcribe ~/Desktop/interview.mp3

This writes: 

~/Desktop/interview.json

Open It in Cuebert

  1. Open Cuebert.
  2. Choose Load transcript.
  3. Select the JSON file, such as interview.json.

If the media file is next to the JSON file, Cuebert should automatically load the audio or video too.

Send Files to an Editor

Send both files: 

interview.json
interview.mp3

After someone starts editing in Cuebert, they may save a interview.cuebert.json file and use that from then on.

Useful Commands

Spanish audio

cuebert-transcribe --language es ~/Desktop/interview.mp3

Replace an existing transcript

cuebert-transcribe --overwrite ~/Desktop/interview.mp3

Several files

cuebert-transcribe ~/Desktop/one.mp3 ~/Desktop/two.mp3

Prepared Kit Layout

The kit folder should look like this on macOS: 

cuebert-transcription-kit/
  cuebert-transcribe
  whisper-cli
  models/
    ggml-large-v3.bin

On Windows, the executable names are cuebert-transcribe.exe and whisper-cli.exe.
